### 编译错误修复汇总

**日期**: 2026-03-07

#### 最新修复 (2026-03-07)

**1. messagecache.cpp - QJsonValueRef 类型转换错误**

**错误**: `'class QJsonValueRef' has no member named 'toVariantMap'; did you mean 'toVariant'?`

**原因**: `QJsonValueRef` 没有 `toVariantMap()` 方法，应该使用 `toVariant().toMap()`

**修复**:
```cpp
// 修复前
messages[msgIt.key()] = msgIt.value().toVariantMap();  // ❌ 不存在的方法

// 修复后
messages[msgIt.key()] = msgIt.value().toVariant().toMap();  // ✅ 正确的方法
```
