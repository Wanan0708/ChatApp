关于 ESChat

致力于为用户提供安全、高效、便捷的即时通讯体验

我们的产品

ESChat 是一款现代化的即时通讯应用，采用先进的技术架构，为用户提供流畅、稳定的通讯服务。 无论是个人聊天还是团队协作，ESChat 都能满足您的需求。

我们采用 WebSocket 技术实现实时消息传输，确保消息即时送达。同时， 通过 JWT 认证和加密传输，保障您的通讯数据安全。

✓ 实时消息传输
✓ 端到端加密
✓ 跨平台支持
✓ 群组聊天
💬

技术架构

采用现代化技术栈，打造卓越性能

🖥️

客户端

基于 Qt/QML 构建，支持 Windows、macOS、Linux 多平台

  • Qt 5.15+
  • QML 2.12
  • C++ 17
⚙️

后端服务

Node.js + Express 构建高性能 API 服务

  • Node.js 18+
  • Express 5
  • WebSocket
🗄️

数据库

PostgreSQL 提供可靠的数据存储

  • PostgreSQL 14+
  • 连接池管理
  • 数据备份
🔒

安全认证

JWT Token 认证，保障用户安全

  • JWT Token
  • bcrypt 加密
  • HTTPS/TLS
